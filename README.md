# Rida Kanwal - Premium Portfolio

A high-performance, accessible, and premium portfolio website built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- **Premium UI**: Off-white/Charcoal/Green aesthetic, smooth marquee animations, and glassmorphism details.
- **Dynamic Content**: Powered by a central data file (`src/data/profile.ts`) for easy updates.
- **SEO Optimized**: Metadata, Sitemap, JSON-LD, OpenGraph support.
- **Performance**: Mobile-first, font optimization, and minimal client-side execution.
- **Features**:
  - Dark/Light Mode
  - Resume Download
  - Contact Form (Server Action)
  - Blog & Projects CMS-lite
  - 20+ Pages generated at build time

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    npm start
    ```

## Content Management

Edit `src/data/profile.ts` to update:
- Profile Information
- Experience & Education
- Projects & Blog Posts
- Services & Skills
- Tools & Software

## Deployment

Deployable to Vercel with zero configuration.
